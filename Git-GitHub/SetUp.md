#### Initialize Git

``` bash
git init --initial-branch=main OR git init -b main # Initialize your new repository and set the name of the default branch to `main`.

git status # Use a `git status` command to show the status of the working tree.

git add # Tell Git to start keeping track of changes in certain files. The technical term is _staging_ these changes.

git commit # After you've staged some changes for commit, you can save your work to a snapshot.

git log # See information about previous commits.
```
